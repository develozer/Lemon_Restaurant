# Restaurant Menu Website

This project is a simple and responsive restaurant menu webpage built as part of a Coursera course.

## Description
The goal of this project was to practice HTML and CSS by designing a clean and user-friendly restaurant menu interface.

## Technologies Used
- HTML5
- CSS3

## Features
- Clean and simple layout
- Responsive design
- Organized menu sections
- Basic styling and typography

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

## Purpose
This project was created for learning and practicing front-end development fundamentals.
